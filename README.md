# Recast.AI - SDK iOS

Recast.AI official SDK in Swift.

## Synospis

 This pod is a Swift wrapper around the [Recast.AI](https://recast.ai) API, and allows you to: 
 * [Analyse your text](https://github.com/RecastAI/SDK-iOS/wiki/Analyse-text)
 * [Manage your conversation](https://github.com/RecastAI/SDK-iOS/wiki/Build-your-bot)

## Requirements

- iOS 10.0+
- Xcode 8.0+

## Installation

RecastAI is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod "RecastAI"
```
 
```swift
 import RecastAI
```
